# 哈奇

哈奇APP主要针对普及业主、家属或租户在社区用于日常服务使用，住户可以在APP完成开门或家居控制等相关指令，同时住户还可以在平台了解物业动态信息、订购物业提供的相关服务，及订购小区周边等商家带来的服务，具体相关服务或功能详见产品结构介绍
